## Tecnologias
* Kotlin
		 
* Activity para host de telas
* Intent Implicita para abrir um app específico
* viewBinding para recuperar components de UI

* ViewGroup
    * ConstraintLayout	
					
* Components de UI
    - TextView
    - EditText
    - Button
    - Drawable para shape button
